🧪 Lab Health Analyzer

A modern web-based tool that helps analyze common lab test results and provides instant insights and recommendations. Built with HTML, CSS, and JavaScript, this project is hackathon-ready and user-friendly.

🚀 Features

✅ User Input: Paste or type lab test values in a structured format (e.g., Test Name = Value).

✅ Smart Analyzer: Detects abnormal vs. normal values for common medical tests.

✅ Instant Results: Displays results as interactive cards with color-coded status.

✅ Recommendations: Provides actionable advice based on test results.

✅ Responsive Design: Works seamlessly on desktop and mobile.

✅ Modern UI: Gradient backgrounds, card-based layout, and smooth hover effects.

📋 Supported Tests

The analyzer currently supports (with medical ranges):

Blood Tests: Platelet Count, Hemoglobin, Hematocrit, RBC Count

Infections: NS1 Antigen (Dengue), Malaria Parasite, Widal Test (Typhoid)

Kidney: Urine Protein

Diabetes: HbA1c / Glycated Hemoglobin

Blood Pressure: Systolic & Diastolic

Cholesterol & Lipids: Total Cholesterol, LDL, HDL, Triglycerides

Liver Function: ALT, AST

🖥️ Usage

Clone this repository or download files.

Open index.html in any browser.

Enter lab values in the textarea (format: Test = Value).

Click "Analyze Report" to see results.

Example Input:

Platelet Count = 120000
NS1 Antigen = 0
Malaria Parasite Count = 250
Widal O titer = 160
Urine Protein mg/dL = 20
A1C = 6.2
Systolic Pressure = 120
Diastolic Pressure = 90
LDL Cholesterol = 170
HDL Cholesterol = 30
Triglycerides = 200
Hemoglobin = 10
Hematocrit = 30
Red Blood Cell Count = 3.0
ALT = 75
AST = 45
